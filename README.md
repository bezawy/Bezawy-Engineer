### Run Locally

To run **Bezawy-Engineer** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/bezawy/Bezawy-Engineer.git
```

Windows:

```bash
git clone https://github.com/bezawy/Bezawy-Engineer.git
```
